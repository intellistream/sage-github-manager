# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release extracted from SAGE project
- Core GitHub Issues management functionality
- Download and sync capabilities
- Statistics and analytics
- AI-powered analysis features
- Team management
- Project organization tools
- Comprehensive CLI interface
- Python API

## [0.1.0] - 2025-01-03

### Added
- First independent release of GitHub Issues Manager
- Extracted from SAGE project's `sage-tools` package
- Standalone configuration system
- Independent CLI commands (`github-manager`, `gh-issues`)
- Comprehensive documentation
- MIT License

### Changed
- Adapted from SAGE-specific paths to `.github-manager` directory
- Updated import statements to use `sage_github` namespace
- Simplified configuration to remove SAGE dependencies
